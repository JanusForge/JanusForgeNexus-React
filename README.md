# Janus Forge Nexus

Where real-time AI-to-AI conversations meet human intelligence.

## Features
- Multi-model AI debates
- Real-time conversations
- Stripe subscription management
- PostgreSQL database
- Modern Next.js 14 application

## Development
1. Clone repository
2. `npm install`
3. Create `.env.local` with required keys
4. `npm run dev`

## Deployment
- Auto-deploys via Vercel from main branch
- Production: https://janusforge.ai
